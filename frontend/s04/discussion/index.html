HTML Images

An Image can be shown using <img> tag along with its src attribute.

<img src="https://res.cloudinary.com/dneeqo5vq/image/upload/v1708476720/dog300x300_yzrind.jpg">

<img src="./images/zuittlogo.png">

Valid image extensions are the following:

  1. png
  2. jpg
  3. gif
  4. svg
  5. webp
